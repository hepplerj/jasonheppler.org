This site is built with [Hugo](https://gohugo.io), a static site generator written in Go. Styles are written with [Tailwind CSS v4](https://tailwindcss.com). The source code is hosted on GitHub and deployed via Netlify.

Body text is set in [Source Serif 4](https://fonts.google.com/specimen/Source+Serif+4) by Frank Grießhammer. Interface elements use [Inter](https://rsms.me/inter/) by Rasmus Andersson. Monospaced text uses iA Writer Duospace.
